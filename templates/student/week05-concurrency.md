# P-O-D-A-R 카드 📇

```text
P (문제) {{1줄 — 예: 재고 1 동시 50요청 음수 재고 -12}}
O (옵션) 비관적 락 / 낙관적 락 / 분산 락
D (결정) {{1줄 — 예: 비관적 락 (경합 잦음, 재시도 비용 ↑)}}
A (행동) {{1줄 — 예: @Lock(PESSIMISTIC_WRITE) + repository 메서드 분리}}
R (결과) {{1줄 — 예: 음수 0건, p99 240ms, 실패율 0%}}
```
