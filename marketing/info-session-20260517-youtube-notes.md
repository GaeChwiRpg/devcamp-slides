### 자주 나올 질문 (Q&A 미리 답 준비)

1. **비전공자도 진짜 따라가나요?**
   → 사전학습 2주로 Git/SQL/Java 기초 맞춤. _완전 초보_ 는 아니고 _Java 문법 이해 + IntelliJ 동작_ 까지는 필요.

2. **AI 정책 — W1~7 안 쓰면 손해 아닌가요?**
   → 써도 됨. 단 _제출 코드 본인이 설명 가능_ 해야 인정. evidence 흔적이 더 중요.

3. **다른 부트캠프와 차이는?**
   → 수료증 X → _PR + 정량 수치 + 이력서_. 채용 담당자가 _링크 한 번_ 으로 검토. (Outcomes 슬라이드)

4. **환불 정말 가능?**
   → 사전학습 전 (5/24 이전) 100% · 이후 주차별 차감 · 7영업일 내 처리.

5. **라이브 모두 참여 가능한 시간대?**
   → 평일 20:00–22:00 (직장인 친화). 월·금·토 필수, 나머지 권장 + 녹화.

6. **Java/Spring 외 가능?**
   → ❌ Java + Spring (Kotlin OK). 미션 채점 정상 작동 안 함.
