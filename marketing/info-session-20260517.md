# 산출물 — 채용 담당자가 보는 GitHub

**수료증이 아닙니다.** 본인 GitHub 에 누적되는:

- 매주 PR 1건+ — 10주 후 10건+ 누적 PR
- 정량 수치 리포트 (응답 ms · QPS · hit rate · before/after)
- AI 검증 흔적 — 프롬프트 로그 + hallucination 잡힌 사례
- 팀 협업 기록 — 9·10주차 4인 팀 PR
- 면접용 이력서 최종본 — bullet 정량화 5건+

> 채용 담당자가 _링크 한 번_ 으로 본인 코드 + 판단 근거 확인.
